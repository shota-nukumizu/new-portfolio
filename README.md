# 余談

**Sass(Syntactically Awesome Style Sheets)**<br>
Rubyで開発されたCSSメタ言語。CSSを効率的に記述できるように開発された言語。長く複雑になってしまう記述でも、わかりやすくシンプルに書けるのが最大の特徴である。**CSSとは違って、条件処理や関数を利用できるのが最大の特徴。**
